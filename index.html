Use Freetronics Experimenters Kit to Build Continuously Changing Color RGB LED

The Idea

When I went through all the parts inside Freetronics Experimenters Kit, and when I saw the RGB LED lying inside the kit, I had this idea that this LED continuously changes its color. I had this image in my mind that the color of the RGB LED just changes and never stop.

Demo Video

Because I used iPhone’s camera to capture the video, and the RGB LED is too bright for the camera(even after I set the maximum value of the color to 60 which is pretty low), the beautiful transition from one color to another can be barely seen in the video.

Video Link: http://youtu.be/MyuLrNblgDo

However, you can download the code from the Github repo which is mentioned below and try it on your own Experimenters Kit after you set up the hardware.

Connect the Hardware

Follow instructions of “Project 7:RGB LED” on page 32 in Arduino Experimenters Kit Project Guide.

Hello, Hexo! Good bye, Octopress!

I am leaving Octopress for Hexo.

What is Hexo?

Hexo is a fast, simple and powerful blog framework. It parses your posts with Markdown or other render engine and generates static files with the beautiful theme. All of these just take seconds.

Hexohexo.io/docs/

So, Octopress?

It basically does the same thing. However, it is another framework.

Why Hexo?

  • Better and clearer documentation. See Hexo’s Doc.
    • Explains everything I need to know.
      • How to setup Hexo.
      • Commands.
      • How Hexo works.
      • How to extend Hexo.
      • etc.
  • Easy installation and configuration, also easy extension.
    • Good documentation.
    • Nice code.
  • I get what I expected.
    • Well, almost. See Pullquote section in “Read More”. Luckily I don’t think I am going to use it often. :D

Why Not Octopress?

  • False documentation.
    • In documentation’s example, everything looks fine, BUT,
    • Listed parameters for Codeblock/Backtick Code Block don’t work with fresh installed Octopress. More details listed in Codeblock section of this post.
  • I thought I would get what all Octopress.org has, I did NOT.
    • The parameters I just mentioned.
    • The theme I had on my octopress was also different from octopress.org.
